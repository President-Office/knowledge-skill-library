# Glossary

这里保存学习开源项目时遇到的术语、缩写、项目内概念和领域词汇。

更底层的工程思想和基础概念放在 [[07-foundations/README]]。Glossary 只做简短术语索引，不承载完整概念解释。

## 术语索引

| 术语 | 含义 | 来源项目 | 备注 |
| --- | --- | --- | --- |
| SOLID | 面向对象设计原则集合 | 多项目 | 详见 [[solid]] |
| Monorepo | 多应用/多包放在同一仓库治理的工程组织方式 | Directus、Payload、Vben | 详见 [[monorepo]] |
| Headless CMS | 内容管理和前端展示解耦的 CMS 范式 | Directus、Payload | 详见 [[headless-cms]] |
| MySQL | 常见通用关系型数据库 | RuoYi/Yudao、JeecgBoot | 详见 [[mysql]] |
| PostgreSQL | 功能完整、扩展能力强的关系型数据库 | Directus、Payload | 详见 [[postgresql]] |
| Redis | 内存键值数据库，常用于缓存和临时状态 | RuoYi/Yudao、Yudao Cloud、JeecgBoot | 详见 [[redis]] |
| OLTP | 面向日常交易、状态变化和一致性约束的数据处理方式 | RuoYi/Yudao、ERPNext、JeecgBoot | 详见 [[oltp]] |
| OLAP | 面向分析、报表、统计和指标看板的数据处理方式 | JeecgBoot、RuoYi/Yudao Report、ClickHouse | 详见 [[olap]] |
