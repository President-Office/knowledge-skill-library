# Practice Labs

这里记录对开源项目的本地运行、调试、二次开发、最小复现和改造实验。

## 适合记录

- 本地启动过程。
- 环境依赖和踩坑。
- 修改一个功能后的验证。
- 最小复现案例。
- 对某个架构点的实验。

## 推荐目录

```text
05-practice-labs/
└── <owner>--<repo>/
    ├── local-setup.md
    ├── add-custom-field.md
    └── plugin-demo.md
```

## 注意

- 不保存密钥、token、服务器密码。
- 不把完整第三方源码复制进知识库。
- 实验代码如果需要长期保留，应放到单独代码仓库，这里只保存说明和结论。
