# Spring Boot

Spring Boot 是 Java 后台应用的主流基础框架，负责应用启动、依赖装配、配置管理、Web 接口、事务、数据访问集成和运行时约定。

## 学习重点

- 自动配置如何降低样板代码。
- Controller、Service、Repository 如何分层。
- 事务、配置、过滤器、拦截器和安全框架如何接入。
