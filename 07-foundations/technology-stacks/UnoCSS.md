# UnoCSS

UnoCSS 是原子化 CSS 引擎，用规则和预设按需生成样式类。

它常用于 Vue/Vite 项目中，减少手写样式文件，让页面通过 class 组合快速表达间距、布局、颜色和响应式规则。

## 学习重点

- 项目使用哪些 preset、shortcut 和 transformer。
- 原子类是否提升一致性，还是导致页面样式散乱。
- UnoCSS 和组件库样式、主题变量如何协同。
