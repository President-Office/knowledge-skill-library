# MCP

MCP 是面向 AI Agent 的工具和上下文协议方向，用来把系统能力以可发现、可调用的工具形式暴露给模型。

在知识库中，MCP 的重点不是追逐具体插件，而是理解 Tool、Schema、Capability、Permission 和 Result 如何让软件系统机器可读。
