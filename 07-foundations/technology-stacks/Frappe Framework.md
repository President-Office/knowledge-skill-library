# Frappe Framework

Frappe Framework 是 ERPNext 的底层全栈框架，围绕 DocType、权限、表单、报表、工作流、事件钩子和 app/site 模型组织业务系统。

它的核心不是普通 MVC 分层，而是元数据驱动业务对象。
