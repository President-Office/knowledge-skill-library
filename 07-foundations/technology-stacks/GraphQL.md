# GraphQL

GraphQL 是一种 API 查询语言和类型系统，允许客户端按对象关系声明需要的数据。

它对 AI Agent 友好的一点在于 schema、类型和关系更容易机器读取，但复杂分析仍要结合合适的数据模型。
