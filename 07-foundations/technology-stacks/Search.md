# Search

Search 是系统提供检索能力的技术和产品功能，可能基于数据库索引、全文索引、搜索引擎或向量检索。

后台系统中的搜索要关注权限过滤、字段权重、排序、分页和与业务状态的关系。

## 学习重点

- 搜索基于数据库 LIKE、全文索引、Elasticsearch，还是外部搜索服务。
- 搜索结果是否遵守数据权限和租户隔离。
- 搜索索引如何同步、重建和回滚。
