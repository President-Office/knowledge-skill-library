# 代码生成

代码生成是根据数据库表、配置、模板或元数据生成基础代码的技术。

它适合处理重复结构明确的内容，例如 Controller、Service、Mapper、VO、列表页、表单页和 API 调用，但不应该替代领域建模和业务规则设计。

## 学习重点

- 输入模型来自数据库表、JSON 配置、在线表单还是领域模型。
- 生成内容是否允许二次编辑，二次生成如何避免覆盖业务修改。
- 生成器和项目规范、权限模型、菜单、路由如何对齐。
