# Next.js

Next.js 是 React 生态全栈框架，常用于页面、服务端渲染、API、路由和全栈应用组织。

在 [[payloadcms--payload]] 中，Next.js 的意义是让后台、CMS 和业务应用可以在同一个 TypeScript 工程内协同。
