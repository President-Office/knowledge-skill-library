# Spring Cloud

Spring Cloud 是 Java 微服务生态，用来处理服务发现、配置、网关、熔断、负载均衡、链路治理等分布式系统问题。

它的价值不是“把单体拆开”，而是在服务边界明确后，为分布式运行提供基础设施。
