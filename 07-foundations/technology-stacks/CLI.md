# CLI

CLI 是命令行入口，用来启动服务、生成代码、迁移数据、运行脚本或管理项目。

在成熟框架里，CLI 往往是“开发体验”和“运行时入口”的交汇点：开发者通过它创建项目、启动服务，部署系统也可能通过它执行迁移、构建和任务。

## 学习重点

- CLI 命令最终调用了哪些核心包或服务启动逻辑。
- CLI 如何读取配置、环境变量和工作目录。
- CLI 是纯开发工具，还是生产环境也依赖的运行入口。
