# Spring Security

Spring Security 是 Spring 生态的认证和授权框架，常用于登录、Token 校验、权限拦截、方法级权限和安全上下文传递。

## 学习重点

- 认证和授权是否分清。
- Token、Session、SSO 和多端登录如何处理。
- 网关、服务和前端路由权限如何协同。
