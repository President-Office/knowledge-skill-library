# Sentinel

Sentinel 是限流、熔断、降级和流量治理组件，常见于国内 Java 微服务项目。

读 Sentinel 时要关注保护的是哪个入口、降级策略是什么、失败时用户看到什么，而不是只看是否引入依赖。
