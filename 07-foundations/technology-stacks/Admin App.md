# Admin App

Admin App 是框架内置或配套的管理端应用，通常负责数据列表、表单、权限配置、用户管理、内容编辑和系统设置。

它和普通业务前端不同：Admin App 更强调可配置性、动态表单、动态路由、权限感知和面向运营人员的后台工作流。

## 学习重点

- 页面是手写、代码生成、配置生成，还是根据元数据运行时渲染。
- Admin App 如何消费 API、权限模型和数据 schema。
- Admin App 和 SDK、插件、主题、扩展点之间如何协作。
