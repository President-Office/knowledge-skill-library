# Spring Cloud Gateway

Spring Cloud Gateway 是 Spring Cloud 体系中的 API 网关组件，常用于路由、鉴权入口、限流、跨域、灰度和统一请求处理。

## 学习重点

- 网关和业务服务的边界。
- 认证信息如何穿过网关传递。
- 路由规则、过滤器、限流和日志如何组织。
