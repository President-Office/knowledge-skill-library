# AI Skills

AI Skills 是把特定任务的知识、步骤、约束、脚本和模板封装给 AI Agent 使用的能力单元。

在开源框架中出现 skills 目录，通常说明项目希望把开发、生成、运维或业务流程的一部分变成可被 AI 调用的结构化资产。

## 学习重点

- skill 描述的是开发辅助，还是系统运行时能力。
- skill 是否包含明确输入、输出、边界和验证步骤。
- skill 与 MCP、CLI、文档、代码生成器之间如何协作。
