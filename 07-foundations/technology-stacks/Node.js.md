# Node.js

Node.js 是 JavaScript / TypeScript 的服务端和工具链运行时，常用于前端构建、全栈框架、后台 API、SDK 和 monorepo 工具。

读 TypeScript 后台平台时，要先确认 Node.js 版本、包管理器和 workspace 约定。
