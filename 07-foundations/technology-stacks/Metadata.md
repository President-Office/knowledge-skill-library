# Metadata

Metadata 是描述数据的数据。它不直接保存业务事实，而是描述字段、关系、权限、展示方式、校验规则、枚举、默认值和扩展行为。

在 AI 时代，metadata 的价值会放大，因为它让系统结构对机器可读：AI Agent 不必猜字段含义、关系和权限，而是能从统一入口读取上下文。

## 学习重点

- metadata 描述了哪些内容：字段、关系、权限、展示、工作流还是业务语义。
- metadata 存在哪里：数据库表、JSON 文件、代码配置还是远程服务。
- metadata 和真实业务数据如何保持一致。
