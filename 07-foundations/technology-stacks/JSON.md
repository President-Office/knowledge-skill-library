# JSON

JSON 是常见的数据交换和配置格式，结构简单，适合表达对象、数组、字符串、数字、布尔值和空值。

在框架源码里，JSON 常用于配置、元数据定义、接口返回、测试样例和前后端数据交换。

## 学习重点

- JSON 文件是静态配置、元数据定义，还是生成产物。
- JSON schema 或类型定义是否约束字段结构。
- JSON 配置与数据库 schema、代码类型之间如何同步。
