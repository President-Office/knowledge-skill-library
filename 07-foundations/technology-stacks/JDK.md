# JDK

JDK 是 Java Development Kit，包含 Java 编译、运行和标准工具链。读 Java 后台项目时，JDK 版本会影响语言特性、依赖版本、构建插件和运行环境。

## 学习重点

- 项目要求的 JDK 版本。
- 框架版本和 JDK 版本是否匹配。
- CI、Docker、生产环境是否使用同一版本线。
