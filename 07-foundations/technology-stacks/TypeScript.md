# TypeScript

TypeScript 是 JavaScript 的类型化扩展，适合大型前端、Node 后端、SDK、monorepo 和类型安全配置。

在 Directus、Payload、Yudao 前端项目中，TypeScript 是理解包边界、API 类型、组件约束和构建链路的重要入口。
