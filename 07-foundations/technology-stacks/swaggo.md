# swaggo

swaggo 是 Go 生态中常见的 Swagger/OpenAPI 文档生成工具链。

它通常通过注释扫描 Go 代码，生成接口文档和可交互的 API 调试页面。

## 学习重点

- API 文档是否由代码注释生成，还是由路由/类型自动推导。
- 注释和真实接口实现是否容易漂移。
- 文档是否纳入登录、权限、错误码和分页约定。
