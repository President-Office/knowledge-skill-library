# Maven

Maven 是 Java 项目的构建、依赖和模块管理工具。企业后台项目常用 Maven 多模块组织 framework、module、api、biz、dal 等边界。

## 学习重点

- 父子 POM 如何管理版本。
- 多模块依赖方向是否清楚。
- 构建产物和运行服务边界是否一致。
