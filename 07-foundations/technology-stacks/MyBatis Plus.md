# MyBatis Plus

MyBatis Plus 是 MyBatis 生态的增强工具，常用于 Java 后台的 Mapper、CRUD、条件构造和代码生成。

它能减少重复数据访问代码，但不能替代 [[sql]]、[[schema]]、[[database-index]] 和 [[database-transaction]] 的理解。
