# 数据权限

数据权限控制用户能看到哪些数据行、哪些组织范围的数据，或哪些字段。

它和功能权限不同：功能权限回答“能不能打开这个功能”，数据权限回答“打开以后能看到哪些数据”。

## 学习重点

- 数据范围来自部门、岗位、租户、角色，还是业务规则。
- SQL 过滤由 AOP、拦截器、ORM 插件、手写查询还是数据库策略实现。
- 数据权限是否会影响导出、统计、报表和异步任务。
